Host Redirect

This file accepts the name of a LAN host in the hashtag portion of the URL.

It uses that hostname to compose a URL for the Graphics Canvas folder on that host. This assumes that this folder exists in the specified standard location on the target host.

The typical url for some host is shown below: 


http://myhost/std/app/canvas

For convenience, all of the known hosts are given in the links below. There are also links to the sandbox locations on the Tower and Lenovo, since these are where initial development took place.

In the event that all else fails, there's a link to the Whiteboard as well.
